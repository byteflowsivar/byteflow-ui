# Changelog - Radio

## [1.0.0] - 2026-02-28

### Añadido
- Versión inicial de los componentes `Radio` y `RadioGroup`.
- Sistema de contexto para manejo de grupos de selección única.
- Soporte para orientaciones `vertical` y `horizontal`.
- Estilizado custom con animaciones para el punto central.
- Definición de tokens CSS para tematización.
