# Changelog - Button

Todos los cambios notables de este componente serán documentados en este archivo.

## [1.0.0] - 2026-02-28

### Añadido
- Versión inicial del componente `Button`.
- Soporte para variantes: `primary`, `secondary`, `ghost`.
- Soporte para tamaños: `sm`, `md`, `lg`.
- Implementación de estado `isLoading` con spinner animado.
- Soporte completo para accesibilidad (Aria attributes).
- Estilización mediante variables CSS con prefijo `bf-`.
